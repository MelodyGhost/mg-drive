## Features

- **Authentication:** Register, Login, Reset password, Update profile
- Support drag and drop file
- Create Folder and subfolder
- Upload file easily
- Easy breadcrumbs navigation
- Proffesional fast UI with upload animation

## Install

```sh
npm install
```
